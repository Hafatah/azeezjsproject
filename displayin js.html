DISPLAY METHOD EXAMPLE
USING DOM MANIPULATION

JavaScript can dynamically create, modify, and delete HTML elements using the Document Object Model (DOM).
This method offers fine-grained control over how JavaScript-generated content is displayed on the page.

id="output"document.getElementById("output")
.innerHTML = "Hello, world!";

USING ALERT()

The alert() method displays a modal dialog with a message and an OK button.
It is often used for simple notifications or alerts.

 alert("helloworld")
USING WEB APIs

Modern web APIs such as Canvas, Web Audio, WebSockets, etc.,
allow for rich and interactive content to be displayed in the browser using JavaScript.
These APIs enable advanced features like drawing graphics, playing audio, and real-time communication.

/ Example using Canvas API
var canvas = document.getElementById("myCanvas");
var ctx = canvas.getContext("2d");
ctx.font = "30px Arial";
ctx.fillText("Hello, world!", 10, 50);

USING PROMPT()

The prompt() method displays a modal dialog with an optional message and an input field for the user to enter data.
It is commonly used to gather user input.

var name = prompt("Enter your name:");
console.log("Hello, " + name + "!");

USING CONFIRM

The confirm() method displays a modal dialog with a message and OK/Cancel buttons.
It returns true if the user clicks OK and false if the user clicks Cancel.

var result = confirm("Are you sure?");
console.log("User confirmed: " + result);

USING CONSOLE OUTPUT

JavaScript code can output information or debug messages to the console using the console.log() method.
This is commonly used for debugging purposes.

console.log("Hello, world!");
