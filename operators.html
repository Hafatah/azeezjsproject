Comparison operators and logical operators are fundamental components of programming languages used to evaluate conditions and make decisions.Let's differentiate between them with copious examples:

OPERATORS EXAMPLE

Comparison Operators: Comparison operators are used to compare values and return a boolean result (True or False).
They are used to compare equality, inequality, greater than, less than, etc.
  • Equality comparison

    • x = 5
    y = 3
    print(x == y) # False
  • Inequality comparison

    • print(x != y) # True
  • Greater than comparison

    • print(x > y) # True
  • Less than comparison

    • print(x < y) # False
  • Greater than or equal to comparison

    • print(x >= y) # True
  • Less than or equal to comparison

    • print(x <= y) # False
    Logical Operators: Logical operators are used to combine or modify the results of comparison operations.
    They include and, or, and not.
  • Using 'and' logical operator
    • a = 5
    b = 10
    print(a > 3 and b < 15) # True (both conditions are True)
  • Using 'or' logical operator

    • c = 20
    d = 25
    print(c == 20 or d > 30) # True (at least one condition is True)
  • Using 'not' logical operator

    • e = 15
    print(not e == 10) # True (the condition is not True)

    In summary, comparison operators are used to compare values, while logical operators are used to combine or modify the results of those comparisons to make more complex conditions.
